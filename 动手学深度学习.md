广播机制要求

1.某个维度大小相等 或其中有一个为1 可以广播

2.两个张量在某个维度大小不同 且都不为1 则无法广播

 PyTorch 和 NumPy 之间进行张量（tensor）与数组（ndarray）的相互转换，并强调了它们之间共享底层内存的特性

```python
A = X.numpy()  # 将 PyTorch 张量 X 转换为 NumPy 数组 A
B = torch.tensor(A)  # 将 NumPy 数组 A 转换为新的 PyTorch 张量 B
type(A), type(B)  # 检查 A 和 B 的类型
```

**`X.numpy()` 的作用**：会共享

x.numpy() x为张量调用numpy转化为Numpy数组A  如果x在cpu上 这个操作不会创建新的数据 而是让A和x共享底层内存 

因此修改A的内容会直接反应到x上

**反之，用 `torch.from_numpy(A)` 将 NumPy 数组转换为张量时，同样共享底层内存**

**`torch.tensor(A)` 的作用**：

- 使用 `torch.tensor(A)` 将 `A` 转换为新的 PyTorch 张量 `B`。**会分配新的内存空间**
- 注意：`torch.tensor()` 创建一个新的张量，它不与 `A` 或原始张量共享内存。
- 这意味着修改 `B` 不会影响 `A`，也不会影响原始张量 `X`。

> **什么是CSV文件格式**
>
> CSV（Comma-Separated Values）文件是一种常用的文件格式，用于存储表格数据
>
> - **数据组织**：数据以行和列的形式存储，每一行表示一条记录，每一列表示一个字段。
> - **分隔符**：字段之间使用逗号（`,`）分隔，某些地区的 CSV 文件可能会使用分号（`;`）或其他分隔符。
> - **文本格式**：CSV 文件是纯文本文件，可以用任何文本编辑器打开。
>
> 示例：
>
> ```
> Name,Age,Gender,Score
> Alice,25,F,88
> Bob,30,M,92
> Charlie,28,M,85
> ```
>
> 在这个示例中：
>
> - 第一行是标题行，包含列名。
> - 后面的每一行是一条记录，对应一组数据。
